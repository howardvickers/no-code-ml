Galvanize Data Science Capstone Project

Machine learning to predict asthma rates

The goal of this project was to use machine learning algorithms to predict asthma rates as influenced by pollutants and socio-economic factors.

Asthma is a chronic inflammatory airway disease. While asthma may be considered in social, economic, cultural, environmental, hormonal, and genetic terms, the focus of this analysis was socio-economic and environmental.

ZNA Health have generously offered support with identifying data sources and with environmental/medical questions.
